# CLAUDE.md

> 이 프로젝트에서 Claude Code가 따를 행동 지침입니다.
> 무엇을 만들 것인가는 ROADMAP.md를 참조합니다.

---

## 1. 프로젝트 컨텍스트

이 프로젝트의 상세 결정사항(사이트 구조, 콘텐츠 톤, 디자인 방향, 누적 운영 정책 등)은 `ROADMAP.md`에 정의되어 있습니다. 다음 중 해당하는 작업일 때 ROADMAP.md를 참조할 것:

- 새 페이지 / 섹션 추가
- 새 Project / Side Quest / Experience 항목 추가
- 디자인 시스템 · 스택 관련 변경
- 콘텐츠 톤이 모호한 경우

단순 버그 수정 · 오타 · 스타일 미세 조정에는 참조 불필요.

**핵심 인덱스 (자주 쓰는 정보):**

- 스택: Astro + Tailwind CSS + Content Collections + TypeScript
- 콘텐츠 위치: `src/content/projects/`, `src/content/side-quests/`, `src/content/experience/`
- 정렬: `featured: true`는 최대 3개 (초과 시 사용자에게 확인)
- 배포: GitHub Pages, 저장소 `lynstory.github.io`
- 폰트: Pretendard (국문) + Inter (영문)
- 포인트 컬러: deep blue 계열 1색

## 3. 금지 표현

사이트 콘텐츠 작성 시 다음 표현은 사용하지 않습니다.

- 의료적 직접 서술: 우울증, 자살, 자해, 정신과, 상담사, 병원, 약물 등
- 가족 관련 사적 사항
- 군대 관련 사적 사항
- 특정 인물의 실명 / 식별 정보

우울증·은둔의 시기는 간접·은유적으로만 표현합니다 (예: "은둔과 자기 이해의 시간", "사람과 삶을 깊이 다시 본 시기"). 자세한 톤 가이드는 ROADMAP.md §3 참조.

## 4. 불확실 시 행동 원칙

- ROADMAP.md에 없는 정보(개인 이력, 사실 관계, 디자인 결정 등)는 추측해서 채우지 말 것
- 불확실하면 작업을 멈추고 사용자에게 질문할 것
- 사용자 정보를 임의로 각색·창작하지 말 것
